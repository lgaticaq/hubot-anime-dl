> A Hubot script to download anime videos

## Installation
```bash
npm i -S hubot-anime-dl
```

add `["hubot-anime-dl"]` to `external-scripts.json`.

## Examples

`hubot anime-dl get <anime> chapter <chapter>` - `Get the link chapter`
